# WPF Website

This project is concerned with the Wetlands Park Friends website, which I have been asked to optimize.

## Project Purpose

The goal of this project is to improve the Wetlands Park Friends website by making it clearer, easier to navigate, and more effective for visitors, members, and donors.

## Project Focus

This project will include work on:

- Website layout and structure
- Header and navigation improvements
- Home page improvements
- About page improvements
- Donate page improvements
- Membership renewal options
- Search engine optimization
- Mobile-friendly design
- Clearer page content
- Improved user experience

## Organization

Wetlands Park Friends supports Clark County Wetlands Park through education, stewardship, special projects, and visitor experiences.
